# Проект Базы Данных "Онлайн-Магазин" (MS SQL Server)

Это мой первый демонстрационный проект базы данных для портфолио. 

## Что внутри проекта:
* **Таблицы:** Клиенты (`Customers`) и Заказы (`Orders`). Настроены связи и автоматическое удаление заказов при удалении клиента.
* **Представления (Views):** Готовый отчет `OrderSummary`, который показывает общую сумму покупок по каждому клиенту.
* **Процедуры:** Имеется процедура `GetCustomerOrders` для быстрого поиска истории заказов конкретного пользователя.
* **Тестовые данные:** В конце файла добавлены строки для проверки работы базы.

## Как запустить:
1. Скопируйте код из файла `schema.sql`.
2. Вставьте его в Microsoft SQL Server Management Studio (SSMS).
3. Нажмите кнопку **Execute** (Выполнить).
